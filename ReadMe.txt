This program was designed to read the sentimentality of an article or Passage.
A URL can be provided for use in extracting the text from said URL and determining the sentimentality

How to use:
to start either paste in an article or type in a URL and click the fetch URL button to retrieve the text from the article
Click the analyze article button to determine the sentimentality of said article.

What was used:
this program uses the requests library, the beautifulsoup library, Textblob, and tkinter.

Requests and Beautifulsoup are used for web scraping.
Textblob is used for determining the sentiment of a URL or Article.
tkinter is used for the gui (graphic user interface).
